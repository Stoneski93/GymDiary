### Getting Started

1. Run `npm install` from root directory

### Run on Android

1. Genymotion
2. Once it's running, in a new terminal tab/window, type `react-native run-android`

### Opening the Debug Menu

- **iOS**: CMD + D
- **Android**: CMD + M
